# Agent-resultaten Kerstmenu 2025

Deze pagina geeft een overzicht van alle resultaten van de inhoudelijke agents die samen het kerstmenu 2025 hebben gemaakt. Dit is bedoeld als "achter de schermen"-overzicht voor iedereen die wil zien hoe het menu is opgebouwd.

Alle links verwijzen naar markdown-bestanden in deze repository. Via GitHub Pages zijn ze rechtstreeks te bekijken in de browser.
