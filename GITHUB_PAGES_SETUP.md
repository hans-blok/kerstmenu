### Wat is GitHub Pages?
GitHub Pages is een gratis dienst die je HTML bestanden online zet, zodat iedereen ze kan bekijken zonder GitHub account. Perfect voor het delen van je kerstmenu!

## ğŸš€ Activatie Stappen

### 1. Open GitHub Repository Settings

1. Ga naar: `https://github.com/<jouw-username>/kerstmenu-2025`
2. Klik op de **Settings** tab (âš™ï¸ icoon, rechtsboven)

### 2. Navigeer naar Pages

1. In het linkermenu, scroll naar beneden
2. Klik op **Pages** (onder "Code and automation")

### 3. Configureer Source

Je ziet nu een pagina met "GitHub Pages" settings:

1. Bij **Source**: Selecteer **Deploy from a branch**
2. Bij **Branch**: 
   - Selecteer **main** (eerste dropdown)
   - Selecteer **/publish** (tweede dropdown)
   - Klik op **Save**

### 4. Wacht op Deployment

- GitHub begint nu met het bouwen van je website
- Dit duurt 1-5 minuten
- De pagina refresht automatisch
- Je ziet een blauwe box met "Your site is ready to be published at..."

### 5. Check je Website

- Klik op de URL in de blauwe/groene box
- Of ga naar: `https://<jouw-username>.github.io/kerstmenu-2025/`
- Je ziet nu het kerstmenu!

## â“ Problemen?

### Website niet zichtbaar na 5 minuten?
- Check of Branch goed is ingesteld (main + /publish)
- Check of er een groene checkmark is bij "Actions" tab
- Probeer een "hard refresh" in je browser (Ctrl+F5)

### Styling ziet er raar uit?
- Check of `style.css` in de publish folder staat
- Doe een hard refresh in je browser (Ctrl+F5)

### 404 Error?
- Check of de URL juist is (let op de username!)
- Check of /publish folder de juiste bestanden bevat
- Wacht nog 2 minuten en probeer opnieuw
